# 뉴스 트렌드 파이프라인에서 데이터 품질을 높이는 전처리 가이드

이 문서는 GitHub 프로젝트의 뉴스 수집 → 정제 → 키워드 추출 → 트렌드 분석 과정에서 **불용어**, **복합명사**, **전처리**가 왜 중요한지 누구나 이해할 수 있도록 예문 중심으로 정리한 1페이지 가이드입니다.

---

## 1. 프로젝트 흐름에서 전처리는 어디에 쓰이나요?

뉴스 트렌드 파이프라인은 보통 아래 흐름으로 동작합니다.

```text
뉴스 수집 → 원문 저장 → 텍스트 전처리 → 키워드/명사 추출 → 트렌드 집계 → 결과 제공
```

여기서 **텍스트 전처리**가 부정확하면 이후 단계가 모두 흔들립니다.
예를 들어 같은 의미의 문장이 서로 다르게 남아 있거나, 의미 없는 단어가 너무 많이 포함되면 잘못된 키워드가 상위 트렌드로 올라올 수 있습니다.

---

## 2. 불용어 제거: 의미 없는 반복 단어 줄이기

**불용어**는 자주 나오지만 분석에는 큰 도움이 되지 않는 단어입니다.
뉴스 데이터에서는 조사, 접속사, 기자 표현, 기사 말미 문구 등이 불용어가 될 수 있습니다.

### 예문

```text
전처리 전:
정부는 오늘 새로운 부동산 대책을 발표했다고 밝혔다.

불용어 제거 후:
정부 오늘 새로운 부동산 대책 발표
```

### 왜 좋아지나요?

`는`, `을`, `했다고`, `밝혔다` 같은 표현은 기사 문장에서는 자연스럽지만, 트렌드 분석에서는 핵심 주제가 아닐 수 있습니다.
불용어를 줄이면 모델이나 분석 로직이 **정부, 부동산, 대책, 발표** 같은 핵심 단어에 더 집중할 수 있습니다.

### GitHub 프로젝트 적용 예

```text
입력 데이터:
삼성전자는 올해 반도체 투자를 확대할 계획이라고 밝혔다.

전처리 결과:
삼성전자 올해 반도체 투자 확대 계획

트렌드 후보:
삼성전자, 반도체, 투자확대
```

## 3. 복합명사 처리: 함께 봐야 의미가 살아나는 단어 묶기

**복합명사**는 여러 단어가 합쳐져 하나의 의미를 만드는 표현입니다.
뉴스 트렌드 분석에서는 복합명사를 잘 묶어야 실제 이슈를 놓치지 않습니다.

### 예문

```text
전처리 전:
한국은행은 기준 금리를 동결했다.

잘못 나눈 결과:
한국 / 은행 / 기준 / 금리 / 동결

더 좋은 결과:
한국은행 / 기준금리 / 동결
```

### 왜 좋아지나요?

`한국`과 `은행`을 따로 보면 국가와 금융기관이 분리되어 의미가 흐려집니다.
`기준`과 `금리`도 따로 보면 핵심 경제 지표인 `기준금리`를 정확히 잡기 어렵습니다.
복합명사를 묶으면 트렌드 결과가 더 자연스러워집니다.

### GitHub 프로젝트 적용 예

```text
입력 데이터:
인공지능 반도체 시장이 빠르게 성장하고 있다.

단순 분리:
인공 / 지능 / 반도체 / 시장 / 성장

복합명사 반영:
인공지능반도체 / 시장성장

트렌드 후보:
인공지능반도체, 시장성장
```

## 4. 기본 전처리: 지저분한 원문을 분석 가능한 문장으로 바꾸기

뉴스 원문에는 HTML 태그, 특수문자, 중복 공백, 이모지, URL, 기자 정보 등이 섞일 수 있습니다.
전처리는 이런 요소를 정리해 데이터 품질을 높이는 작업입니다.

### 예문

```text
전처리 전:
[속보] 정부, AI 산업 육성안 발표!!! 자세한 내용은 https://news.example.com 참조 <br> 김OO 기자

1단계: HTML, URL 제거
[속보] 정부, AI 산업 육성안 발표!!! 자세한 내용은 참조 김OO 기자

2단계: 특수문자와 불필요 표현 정리
정부 AI 산업 육성안 발표 자세한 내용 참조 김OO 기자

3단계: 불용어/기자 정보 제거
정부 AI 산업 육성안 발표

4단계: 복합명사 반영
정부 AI산업 육성안 발표
```

최종적으로 분석에 사용할 문장은 다음처럼 짧고 명확해집니다.

```text
정부 AI산업 육성안 발표
```

## 5. 전처리 전후 비교

| 처리 항목 | 전처리 전 | 전처리 후 | 좋아지는 점 |
|---|---|---|---|
| 불용어 제거 | 정부는 대책을 발표했다고 밝혔다 | 정부 대책 발표 | 핵심 단어가 잘 보임 |
| 복합명사 처리 | 기준 / 금리 / 인상 | 기준금리 / 인상 | 실제 경제 이슈를 정확히 잡음 |
| 특수문자 제거 | AI 산업!!! 급성장??? | AI 산업 급성장 | 불필요한 기호로 인한 오류 감소 |
| URL 제거 | 자세히 보기 https://... | 자세히 보기 | 분석과 무관한 문자열 제거 |
| 기자 정보 제거 | 홍길동 기자, 입력 2026.04.27 | 본문 내용만 유지 | 기사 본문 중심 분석 가능 |

---

## 6. 한 문장으로 정리

뉴스 트렌드 분석에서 좋은 전처리는 **불필요한 표현은 줄이고, 함께 봐야 하는 단어는 묶고, 지저분한 원문은 깨끗하게 정리해서 실제 이슈가 잘 보이게 만드는 과정**입니다.

```text
좋은 전처리 = 핵심 키워드는 살리고 + 노이즈는 줄이고 + 의미 단위는 보존하는 것
```

이 과정이 잘 되어야 GitHub 프로젝트의 키워드 추출, 트렌드 집계, 검색 결과 품질이 함께 좋아집니다.
